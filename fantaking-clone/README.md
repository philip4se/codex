# 판타지 킹 - 판타지 축구 게임

판타킹 for Kakao와 유사한 판타지 축구 시뮬레이션 게임입니다.

## 게임 소개

실제 판타킹과 동일한 구조를 가진 웹 기반 축구 카드 게임입니다. 선수를 수집하고, 팀을 구성하며, 다른 팀과 경쟁하여 리그 순위를 올리세요!

## 주요 기능

### 1. 선수 카드 시스템
- 4가지 등급: 일반(⭐), 레어(⭐⭐), 에픽(⭐⭐⭐), 전설(⭐⭐⭐⭐)
- 다양한 능력치: 페이스, 슈팅, 패스, 드리블, 수비, 피지컬
- 포지션별 특화 능력치
- 선수 레벨 시스템

### 2. 팀 구성
- 5가지 포메이션: 4-4-2, 4-3-3, 3-5-2, 4-2-3-1, 3-4-3
- 실제 축구장을 본떠 만든 비주얼 인터페이스
- 포지션별 선수 배치
- 팀 파워 계산 시스템

### 3. 카드 뽑기 (가챠)
- 일반 뽑기: 골드로 구매
  - 일반: 60%
  - 레어: 30%
  - 에픽: 10%
- 프리미엄 뽑기: 다이아로 구매
  - 레어: 50%
  - 에픽: 35%
  - 전설: 15%
- 10연차 할인 (10% 할인)

### 4. 선수 강화
- 골드를 사용한 선수 레벨업
- 레벨 상승 시 능력치 향상
- 전체 파워 증가

### 5. 리그 시스템
- AI 상대팀과의 경기
- 실시간 경기 시뮬레이션
- 승/무/패에 따른 포인트 획득
- 경기 통계 (점유율, 슈팅, 코너킥 등)
- 경기 보상 (골드, 다이아, 경험치)

### 6. 데이터 저장
- 자동 저장 (30초마다)
- 로컬스토리지 활용
- 진행 상황 영구 보존

## 게임 시작하기

1. `index.html` 파일을 웹 브라우저로 열기
2. 초기 선수 5명으로 시작
3. 스쿼드 구성 후 경기 시작
4. 카드 뽑기로 더 강한 선수 획득
5. 선수 강화로 팀 파워 향상
6. 리그에서 승리하며 순위 상승

## 파일 구조


## 게임 플레이 팁

1. **초반 전략**
   - 초기 골드로 일반 뽑기 여러 번 수행
   - 포지션별 최소 1명씩 확보
   - 스쿼드 11명 완성 후 경기 시작

2. **효율적인 강화**
   - 레어 이상 등급 선수 우선 강화
   - 공격수와 미드필더 먼저 강화 추천
   - 골키퍼는 후순위

3. **재화 관리**
   - 골드: 일반 뽑기 및 선수 강화
   - 다이아: 프리미엄 뽑기 (전설 선수 획득 기회)
   - 경기 승리로 꾸준히 재화 확보

4. **팀 구성**
   - 4-3-3: 공격적인 플레이
   - 4-4-2: 균형잡힌 플레이
   - 3-5-2: 미드필드 장악
