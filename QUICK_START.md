# 🚀 빠른 시작 가이드

## 단 4줄로 시작하기!

```bash
npm install
npx prisma db push
npx prisma generate
npm run dev
```

그리고 브라우저에서 http://localhost:3000 접속!

## 📸 화면 구성

### 메인 페이지
- 설문 의뢰자 / 참여자 선택
- 회원가입 / 로그인

### 설문 의뢰자 대시보드
- 내가 등록한 설문 목록
- 새 설문 등록 버튼
- 응답자 수 확인

### 설문 참여자 대시보드
- 나에게 맞는 설문 목록
- 보상 정보 표시
- 참여 상태 (참여 가능/진행중/완료)

### 설문 등록 페이지
- 기본 정보 (제목, 설명, 구글 시트 URL)
- 타겟 설정 (나이, 지역, 직업)
- 보상 설정 (종류, 내용, 조건)

### 응답자 관리 페이지
- 완료된 응답자 목록 테이블
- 연락처, 이메일, 개인정보 확인
- CSV 다운로드 버튼

## 💡 팁

1. **회원가입 시 정확한 정보 입력**
   - 지역, 나이, 직업 정보가 설문 매칭에 사용됩니다

2. **구글 시트 공유 설정**
   - 설문 URL은 "링크가 있는 모든 사용자" 권한 필요

3. **보상 정보는 명확하게**
   - 참여자가 쉽게 이해할 수 있도록 작성

4. **데이터베이스 관리**
   ```bash
   npx prisma studio  # 브라우저에서 DB 내용 확인
   ```

## 🎯 첫 번째 설문 만들기

1. 회원가입/로그인
2. "설문 의뢰하기" 클릭
3. "새 설문 등록" 클릭
4. 설문 정보 입력:
   - 제목: "2024년 소비 트렌드 조사"
   - 구글 시트: https://docs.google.com/spreadsheets/d/your-sheet-id
   - 타겟: 20-30세, 서울/경기 지역
   - 보상: 스타벅스 기프티콘
5. 등록 완료!

이제 조건에 맞는 참여자들에게 설문이 노출됩니다! 🎉
