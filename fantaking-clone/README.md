# 판타킹 for Kakao - RTS 전략 게임

2013년에 출시되어 많은 유저들의 사랑을 받았던 실시간 전략 시뮬레이션 게임 **판타킹 for Kakao**를 웹으로 재현한 프로젝트입니다.

## 🎮 게임 소개

판타킹은 실시간 전략 시뮬레이션(RTS)에 롤플레잉(RPG)과 디펜스 장르의 재미를 더한 게임입니다. 자원을 획득하고, 유닛을 생산하며, 적진의 거점을 점령하여 승리를 쟁취하세요!

## ⚔️ 주요 특징

### 1. 세 가지 종족 시스템
- **인간 (Human)** ⚔️
  - 강력한 공격력과 높은 체력
  - 정면 돌파에 특화
  - 추천: 초보자

- **엘프 (Elf)** 🏹
  - 빠른 이동속도와 높은 민첩성
  - 치명타 특화
  - 추천: 기동전 선호

- **언데드 (Undead)** 💀
  - 강력한 마법 능력
  - 특수 능력과 소환 마법
  - 추천: 마법 활용 고수

### 2. 거점 점령 시스템
- **3단계 거점 점령**: 외성 → 내성 → 내부
- 거점을 점령하면 자동으로 유닛 생산
- 전략적인 거점 선택이 승패를 좌우

### 3. 다양한 유닛
각 종족마다 4가지 고유 유닛:
- 기본 유닛부터 최상급 유닛까지
- 각 유닛별 고유한 능력치와 특성
- 유닛 조합을 통한 전략적 플레이

### 4. 강력한 마법 시스템
- **인간**: 치유, 광폭화, 보호막
- **엘프**: 화살 폭격, 질주, 자연의 축복
- **언데드**: 암흑 화살, 역병, 소환, 생명력 흡수

### 5. 다양한 게임 모드
- **캠페인**: 6개 스테이지, 난이도별 도전
- **침공**: 다른 플레이어 자원 약탈
- **서바이벌**: 끝없는 적 방어
- **레이드**: 강력한 보스 처치

### 6. 아이템 & 합성 시스템
- 1000여 개 이상의 아이템
- 5단계 희귀도 (일반 ~ 전설)
- 아이템 합성으로 더 강력한 장비 제작

### 7. 데이터 저장
- 자동 저장 (30초마다)
- LocalStorage를 통한 영구 저장
- 언제든 이어서 플레이 가능

## 🎯 게임 플레이 방법

### 시작하기
1. 게임을 시작하면 종족 선택 화면이 나타납니다
2. 세 종족 중 하나를 선택하세요 (변경 시 진행 상황 초기화)
3. 초보자에게는 **인간** 종족을 추천합니다

### 캠페인 모드
1. **캠페인** 탭을 클릭합니다
2. 스테이지를 선택하세요 (처음엔 1스테이지만 해제)
3. 전투가 시작되면:
   - 거점을 클릭하여 점령
   - 마법 버튼을 클릭하여 시전
   - 유닛들이 자동으로 적을 공격
4. 적의 본거지를 파괴하면 승리!

### 전투 팁
- **초반**: 중립 거점을 빠르게 점령하세요
- **중반**: 거점에서 생산되는 유닛으로 병력 우위 확보
- **후반**: 마법을 적절히 사용하여 적 진영을 압박
- 거점은 3단계로 나뉘므로 끈기있게 공격하세요

### 마법 활용
- 마법은 마나를 소모합니다 (자동 회복)
- 각 마법은 재사용 대기시간이 있습니다
- 위기 상황에서 치유나 버프 마법을 사용하세요
- 언데드의 소환 마법은 병력이 부족할 때 유용합니다

### 자원 관리
- **골드** 💰: 유닛 생산, 아이템 구매에 사용
- **보석** 💎: 고급 아이템, 특별 능력에 사용
- 전투 승리로 자원 획득
- 침공 모드로 더 많은 자원 획득 가능

## 💡 FAQ

**Q: 종족을 바꿀 수 있나요?**
A: 네, 가능합니다. 단, 모든 진행 상황이 초기화됩니다.

**Q: 데이터가 저장되나요?**
A: 네, 30초마다 자동 저장되며 브라우저의 LocalStorage에 저장됩니다.

**Q: 가장 강한 종족은?**
A: 각 종족마다 장단점이 있습니다. 플레이 스타일에 맞는 종족을 선택하세요.

**Q: 레벨업은 어떻게 하나요?**
A: 전투에서 경험치를 획득하여 자동으로 레벨업됩니다.

**Q: 아이템은 어디서 얻나요?**
A: 전투 승리 시 확률적으로 드롭됩니다. 레이드에서 더 높은 확률로 획득 가능합니다.
